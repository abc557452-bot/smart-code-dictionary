# قاموس الكود الذكي

قاموس برمجي بسيط للبحث عن كودات جاهزة مع شرحها، مناسب لتعليم المبتدئين والمتوسطين في البرمجة.

## المميزات
- البحث عن الكودات الجاهزة
- شرح بالانجليزية
- شرح بالعربية
- أمثلة قابلة للتجربة مباشرة

## التقنيات
- HTML
- CSS
- JavaScript

## كيفية الاستخدام
1. اكتب اسم المصطلح أو الكود في مربع البحث.
2. اضغط زر البحث أو اكتب مباشرة.
3. ستظهر النتيجة مع الكود والشرح بالإنجليزية والعربية.
